Acrotech Level Meter Control

Version 1.4


The Acrotech Level Meter Control can be used to display the level of any realtime data such as a audio stream level or it can be used as progress bar. It is lightweight and has a fast response time. Colors can be set for high, mid and low levels. A peak indicator is also displayed. The control does not have any dependancies except that it uses the standard VB Timer control.

The following table gives a description of all non-standard properties of the Level Meter Control listed in alphabetical order.


PropertyTypeDescription
BorderBooleanReturns or sets whether a 3D border to drawn around the level meter.
GradientBooleanReturns or sets if the color changes between levels are smooth.
HiColorLong (Color)Returns or sets the color to be used when the level value exceeds the HiLevel property.
HiLevelIntegerReturns or sets the level value beyond which the control will draw the level using the HiColor property.
HorizontalBooleanReturns or sets if the level is displayed horizontally.
LevelIntegerReturns or sets the current level to be displayed. This is the default property. Must be set between 0 to MaxLevel inclusive.
LowColorLong (Color)Returns or sets the color to be used when the level value is below the MidLevel property.
MaxLevelIntegerReturns or sets the maximum level that the level meter displays. The minimum level possible is always 0.
MidColorLong (Color)Returns or sets the color to be used when the level value is above the MidLevel property but below the HiLevel property.
MidLevelIntegerReturns or sets the level value beyond which the control will draw the level using the MidColor property.
PeakDelayIntegerReturns or sets the delay before which the peak indicator starts falling.
ReverseBooleanReturns or sets if the level meter is reversed.
ShowPeakBooleanReturns or sets if the peak indicator should be drawn.
SolidBooleanReturns or sets if the control is drawn using a solid color instead of lines.
