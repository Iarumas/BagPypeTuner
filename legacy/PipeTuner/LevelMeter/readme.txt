Acrotech Level Meter Control V1.4


Installation Instructions
=========================


1) Copy the file levelm.ocx to the Windows System folder or any other folder.

2) Select Run from the Start menu and type in:

   regsvr32 [control path]

   For example, if you had copied the control to the windows system folder, you 
   would type:

   regsvr32 c:\windows\system32\levelm.ocx


Note: your Windows system path may be different from the path mentioned above.
